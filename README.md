# Microservices Clean Architecture

A  Sample dot net core Project  using Microservices Clean Architecture

# Require
 
1. Visual Studio 2019
2. .NET Core 3.1
3. MsSql Server 2017

# Features
1. Web Api
2. Code First Migraton
3. CommandStack (Mediator)
4. QueryStack
5. Shared Functions
6. EntityFrameworkCore
7. SOLID Principles
8. Design Patterns
9. Swagger
